* 題解
  一個命令行接口哭了
  她說大家都不理她
  她讓我給她寫一篇辯白
* 源起
  * 因原始初等而在初期的計算機上興起
* 發展
  * 與圖形界面的競爭
  * 對比典型的程序
    可以發現其高效性
  * 不必把不用的性狀和功能展示於菜單
* 現狀
  * 因最簡單而被保留
  * 可腳本化
* 批評
  * 某些命令行程序的難用不在於命令行本身的不好
    而在於這些程序設計的問題
